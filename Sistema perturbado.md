### Contenido principal

```ad-Formal
Dado un sistema $Ax = b$, la perturbación de un sistema se puede escribir como
$$(A+E) \hat x = (b+f),$$
donde pasamos a definir $\hat A := A+E$, y $\hat b := b+f$.
```
